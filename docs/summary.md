# docs Summary

- Centralized documentation hub covering every lifecycle stage: installation, configuration, usage, troubleshooting, and advanced topics.
- Getting started materials include `quick-start-guide.md`, `installation-guide.md`, and `daily-usage.md` for new operators.
- Configuration references such as `environment-configuration.md`, `env-variables-reference.md`, and `logging-control.md` explain `.env` keys and logging tweaks.
- API & feature docs (`api-usage.md`, `openai-compatibility.md`, `client-examples.md`, `script_injection_guide.md`, `streaming-modes.md`, `webui-guide.md`) detail how to integrate clients, scripts, and the Web UI.
- Developer-leaning files (`architecture-guide.md`, `development-guide.md`, `dependency-versions.md`, `ui_state_management.md`) describe the modular design and contribution workflow.
